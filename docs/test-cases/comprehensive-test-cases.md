### 1.1 登录功能

#### 1.1.1 正常登录流程
**测试用例ID**: AUTH-001
**测试标题**: 管理员正常登录
**前置条件**:
- 系统已部署并可访问
- 存在有效的管理员账户

**测试步骤**:
1. 访问 `/admin/login` 页面
2. 输入正确的用户名 `admin`
3. 输入正确的密码 `Admin123!`
4. 点击"登录"按钮

**预期结果**:
- 页面跳转到 `/admin/dashboard`
- 显示欢迎信息
